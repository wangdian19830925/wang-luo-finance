# 测试报告 v158

## 版本信息

- **版本号**：v158
- **日期**：2026-06-29
- **变更类型**：UI/UX 优化 + 宏观趋势增强

## 变更内容

### 1. 退休曲线纵轴默认 -2%~5%，动态扩展

**需求**：CPI 和年化收益的纵轴默认为 -2% 到 5%，数据超出范围时再动态扩展。

**实现**：
- `CURVE_CONFIG` 中 `inflation` / `investmentReturn` 的 `min` / `max` 已设为 `-2` / `5`。
- `_getCurveEditorRange()` 读取曲线实际值，当数据最小值 < -2 或最大值 > 5 时，向对应方向取整扩展；否则保持默认范围。
- `_renderRetirementCurveEditor()` 使用动态范围绘制网格、刻度与标签。

**修改文件**：`js/app.js`

### 2. 额外收支按钮对齐与图例优化

**需求**：
- 「额外收支」按钮稍微下移，与「房贷处理方式」单选项视觉对齐。
- 按钮长度缩短一半，靠左。
- 有数据时，图例依次显示在按钮右侧；数据过多排不下时隐藏最旧的记录。

**实现**：
- `css/style.css` 中 `.retirement-extra-btn` 宽度改为 50%，`.retirement-mortgage-right` 增加 `padding-top: 4px` 并对齐。
- `_updateExtraTransactionBadge()` 将记录按年份从新到旧排序，生成带颜色圆点的图例项；父容器 `overflow:hidden`，最右侧（最旧）项优先被截断。

**修改文件**：`index.html`、`css/style.css`、`js/app.js`

### 3. 退休曲线重置按钮文案

**需求**：「恢复宏观推荐」改为「使用机构预测」。

**实现**：`index.html` 中两个曲线卡片的重置按钮文案已修改；`_applyMacroCurve()` 的 toast 文案同步改为「已使用机构预测」。

**修改文件**：`index.html`、`js/app.js`

### 4. 宏观趋势页布局调整

**需求**：
- 移除「退休计算推荐曲线」卡片。
- 「应用到退休计算」改为「应用」。
- 按钮放在「CPI 走势」和「利率走势」图表后面。

**实现**：
- `index.html` 删除推荐曲线卡片，新增「预测参数」操作卡片，将 `应用` 按钮置于 CPI/利率图表之后。
- `renderMacroTrendsPage()` 不再渲染只读推荐曲线。
- `_bindMacroTrendsActions()` 将按钮 ID 从 `applyMacroCurveToRetirementBtn` 改为 `applyMacroToRetirementBtn`。

**修改文件**：`index.html`、`js/app.js`

### 5. 趋势图增加坐标轴、刻度与数据

**需求**：所有趋势图增加横轴、纵轴图例和刻度数据。

**实现**：重写 `_renderMacroLineChart()`：
- 默认范围 `defaultMin=-2`、`defaultMax=5`，支持动态扩展。
- 绘制水平网格线、Y 轴刻度标签、X 轴刻度标签。
- 添加 X/Y 轴标题（时间、指标名称与单位）。
- 历史数据用实线，预测数据用虚线，并用虚线分隔历史与预测。
- 在数据点上绘制小圆点，但控制密度避免重叠。

**修改文件**：`js/app.js`、`css/style.css`

### 6. 每日新闻卡片

**需求**：在宏观趋势中增加「每日新闻」卡片，获取宏观经济新闻（全球股市、汇市、期货、主要经济体数据发布）。

**实现**：
- `index.html` 新增「每日新闻」卡片，含 `macroNewsBody` 容器。
- 新增 `_fetchMacroNews()`、`_renderMacroNews()`、`_classifyNewsTag()`、`_formatNewsNewsTime()`。
- 通过 `rss2json.com` 代理抓取 CNBC 财经 RSS，按标题/描述关键词分类为「股市」「汇市」「期货」「宏观」「财经」，并展示相对发布时间。
- 新闻列表缓存于 `_macroNewsCache`，避免重复请求。

**修改文件**：`index.html`、`css/style.css`、`js/app.js`
