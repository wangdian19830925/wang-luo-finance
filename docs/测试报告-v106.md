## 变更说明

### Bug 修复

- **教育支出为 0 的 bug**：v105 将参数从硬编码 2035 年改为"教育支出结束年龄"（默认 22 岁），但 `currentAge=43`，导致 `22-43=-21`，`educationYears=0`，教育支出全部为 0，资金消耗更快，曲线异常
- **修复方式**：改回"教育支出结束年份"，默认 2035 年；计算逻辑改为 `educationYears = educationEndYear - currentYear`
