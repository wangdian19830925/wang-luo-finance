## 关于绿色/红色区域正确性分析

当前区域填充逻辑：
- 使用 `linearGradient` 从顶部(绿)到底部(红)
- 零线(y=py(0))处是分界点：零线以上绿色渐变→透明，零线以下透明→红色渐变
- 曲线下方到零线的闭合路径填充此渐变

**结论：逻辑正确**
- 🟢 绿色区域 = 可投资产 ≥ 0 的部分（资产为正）
- 🔴 红色区域 = 可投资产 < 0 的部分（资金缺口/负债）
- 当曲线全部在零线以上时，整个区域呈现绿色；当某年份 endBalance < 0 时，该点下方出现红色
