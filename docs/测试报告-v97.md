## 4. 已知问题与风险

| 问题 | 级别 | 说明 |
|------|------|------|
| 基本养老金数据来自截图 | 中 | 已通过截图录入王典最新数据，需后续通过国家社保平台核对 |
| 配偶数据为假设 | 中 | 配偶当前按王典相同数据计算，需补充其真实社保数据 |
| 缴费指数/社平工资为简化假设 | 中 | 缴费指数固定 1.5，社平工资固定 12,000，后续可替换为真实值 |
| 投资收益率/通胀率为默认值 | 低 | 用户可在页面参数中实时调整 |
| 企业年金收益率使用截图 2026 年数据 | 低 | 历史长期平均约 5.13%，可后续增加参数调整 |
| 按月还房贷使用贷款数据估算 | 低 | 优先读取 `monthlyPayment`/`remainingMonths`，缺失时按 endDate/标准公式估算 |
| RSU/保险现金价值今日不计入 | 低 | 符合"今天退休"场景设定 |
