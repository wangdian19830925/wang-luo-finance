## 修复内容

### 清空本地数据操作增加密码保护

**问题**：`clearAllLocalData()` 直接执行 `Storage.clearAllData()`，无需密码验证。任何人拿到设备就能清空所有本地财务数据。

**修复**：将清空操作包裹在 `requirePassword()` 中。执行流程变为：
1. 弹出确认弹窗（"确定清空所有本地数据？"）
2. 如果密码保护已开启，弹出密码验证框
3. 密码验证通过后才执行清空
4. 密码保护未开启时直接执行（与现有行为一致）

**修改文件**：`js/app.js` — `clearAllLocalData()` 方法
